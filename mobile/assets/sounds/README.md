# Ambient Sound Files

Place the following `.mp3` files in this directory:

| Filename        | Scene       | Recommended source                          |
|-----------------|-------------|---------------------------------------------|
| `beach.mp3`     | Beach       | Freesound.org · search "ocean waves loop"   |
| `mountains.mp3` | Mountains   | Freesound.org · search "wind mountain loop" |
| `forest.mp3`    | Forest      | Freesound.org · search "forest birds loop"  |
| `fireplace.mp3` | Fireplace   | Freesound.org · search "fireplace crackling"|
| `rain.mp3`      | Rain        | Freesound.org · search "rain indoor loop"   |
| `nightsky.mp3`  | Night Sky   | Freesound.org · search "crickets night loop"|

All files must be loopable `.mp3`. 30–60 seconds is sufficient since expo-av loops them.
Royalty-free sources: freesound.org (CC0) · pixabay.com/music · zapsplat.com
