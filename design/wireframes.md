## DESIGN PRINCIPLES

  1. Never use red for anything except genuine crisis/emergency UI
  2. Mood states have their own colors вҖ” never label them as "bad" or "good"
  3. Every screen accessible with one thumb (bottom nav + large tap targets)
  4. Journal and cycle log entries are editable up to 48h later
  5. No streaks that punish missing a day вҖ” Equi celebrates returning, not consistency
  6. All community posts anonymous by default вҖ” opt-in to add a display name
  7. No algorithmic feed вҖ” community is chronological only
  8. Psychiatrist data is never shared without explicit one-time consent per sharing
  9. Export always available вҖ” your data is yours
  10. Offline-first вҖ” core features work without internet
